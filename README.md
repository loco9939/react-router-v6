# 코드 스플리팅(Code Splitting)

React 페이지 컴포넌트를 페이지 경로 마다 동적으로 호출할 수 있도록 코드 스플리팅을 구현합니다.

- [ ] React.Suspense, React.lazy() 사용법
- [ ] 페이지 컴포넌트 코드 스플리팅

| URL 경로       | 컴포넌트  |
| -------------- | --------- |
| /              | Home      |
| /books         | Books     |
| /books/:id     | Book      |
| /consumers     | Consumers |
| /consumers/:id | Consumer  |
