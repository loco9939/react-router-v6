# URL 매개변수

React Router가 제공하는 `useParmas` 훅을 사용해 URL 매개변수를 가져와 페이지를 렌더링합니다.

- [ ] 도서 목록 조회 페이지 구성
- [ ] 국제도서표준번호(ISBN)로 접속 가능한 도서 개별 페이지 구성
- [ ] 고유번호(ID)로 접속 가능한 고객 정보 페이지 구성

| URL 경로      | 컴포넌트 |
| ------------- | -------- |
| /             | Home     |
| /books        | Books    |
| /book/:isbn   | Book     |
| /consumer/:id | Consumer |
